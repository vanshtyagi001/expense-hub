# AI Usage Log

## Methodology
The agentic development sequence leaned heavily on executing complex logic layers sequentially rather than monolithically.

1. **Schema Instantiation**: AI was used to correctly orchestrate relational mappings joining Users -> GroupMembers -> Expenses -> Splits -> Settlements, alongside the specific architecture of `importAnomalies`.
2. **Algorithm Crafting**: AI provided the `minimizeTransactions` greedy vector matching for the Debt Simplification Engine perfectly resolving the graph optimization task without importing excessive math libraries.
3. **Data Normalization**: AI generated deep RegEx + `date-fns` parsers resilient to random data inputs (e.g. `05/03/2026`, `Mar 14`, `2026-04-01`).
4. **UI Scaffolding**: Leveraged heavily to implement robust accessible Radix/Lucide visual standards quickly so that functional algorithms remained the primary focus of development constraints.
